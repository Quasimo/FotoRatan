The Best Image Gallery Application Host on Google App Engine

Build Your Own Simple Beautiful Image Gallery By FotoRatan (图藤)

FotoRatan(图藤) is an open source software package for creating and supporting image hosting. It's written in Python and running on Google App Engine, which is a free and robust cloud hosting infrastructure.

FotoRatan.appspot.com, also a demo for FotoRatan. It shows Some Friends' high-quality original work. If you are interested in the image of your work published here, please send to: linuximo@gmail.com.

Of course, you can use this tool, based on the Google App Engine cloud computing systems to build your own album, please see: Plans and Pricing

FotoRatan is based on GAE image hosting program diabloimage created, diabloimage under GNU Lesser General Public License, the author is benmao.

You can find the project in github.
